# Sequoia-CLI
红杉工作室自用项目脚手架。

## 安装
```bash
$ npm install -g @sequoia/cli
# or

$ yarn global add @sequoia/cli
```

## 使用
```bash
# 创建项目
$ sequoia create projectName

# 更新脚手架
$ sequoia upgrade

# 查看项目模板列表
$ sequoia list

# 查看当前版本
$ sequoia -V
```

## 提供模版
- uni-app-uview
- express
- vue2
- vue3

## 查看帮助
```bash
$ sequoia -h
```
